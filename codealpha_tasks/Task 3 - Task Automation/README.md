# Task Automation - File Renamer 📂

## Description
The Task Automation project is a Python application developed as part of the CodeAlpha Python Programming Internship. This project automates the process of renaming text files in a folder. The user provides a folder path, and the program automatically finds all `.txt` files and renames them sequentially.

## Features
- Accepts a folder path from the user
- Checks whether the folder exists
- Finds all `.txt` files in the folder
- Automatically renames files in sequence
- Displays old and new file names
- Counts the total number of renamed files
- Handles invalid folder paths
- Allows the user to automate multiple folders
