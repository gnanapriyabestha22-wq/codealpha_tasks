# Hangman Game 🎮

## Description
The Hangman Game is a simple word-guessing game developed using Python. The program randomly selects a word from a predefined list, and the player has to guess the word one letter at a time. The player gets a limited number of incorrect attempts to guess the word.

## Features
- Random word selection
- Displays blanks for unguessed letters
- Accepts one letter at a time
- Checks for duplicate guesses
- Validates user input
- Limited number of attempts
- Displays win or lose messages
- Play Again option

## Technologies Used
- Python
- Random Module
- Loops
- Conditional Statements
- Lists
- Functions

## How to Run the Project
1. Download or clone this repository.
2. Open the project folder.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash
python Hangman_Game.py
```
