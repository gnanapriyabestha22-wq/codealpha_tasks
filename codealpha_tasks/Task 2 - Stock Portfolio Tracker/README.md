# Stock Portfolio Tracker 📈

## Description
The Stock Portfolio Tracker is a Python application developed as part of the CodeAlpha Python Programming Internship. The program allows users to enter stock names and the number of shares they own and then calculates the total value of their investment portfolio using predefined stock prices.

## Features
- Displays a list of available stocks and their prices
- Accepts stock names and quantities from the user
- Calculates the value of each stock investment
- Calculates the total portfolio value
- Handles invalid stock names
- Validates user input
- Supports multiple portfolio calculations
- Displays portfolio details in a formatted table

## Technologies Used
- Python
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling
- Formatted Output

1. Clone or download this repository.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

```bash
python Stock_Portfolio_Tracker.py
```
